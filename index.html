🔒 Privasi & Keamanan

✓
Video kamera tidak pernah diupload.
Semua face & hand tracking diproses 100% di browser kamu via face-api.js + MediaPipe (TensorFlow.js).
✓
Wajah dianalisa in-memory.
Tidak ada frame yang disimpan ke disk atau dikirim ke server. Hilang saat reload.
✓
Yang dikirim ke server: hanya nama + skor.
Itu pun via RPC submit_score dengan validasi server-side (regex name, range score 0–5000).
✓
IP kamu di-hash SHA-256 + salt.
Disimpan sebagai hash 64-char, BUKAN IP asli. Anonim — admin pun tidak bisa reverse hash ke IP.
✓
Rate limit 10 submit/menit/IP.
Mencegah spam abuse di leaderboard.
✓
Open source MIT.
Audit code sendiri di GitHub.
